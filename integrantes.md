Descripción del Proyecto
Programa básico en lenguaje C que gestiona y calcula el promedio académico de un grupo de estudiantes.  
Solicita la cantidad de alumnos y valida que cada calificación esté en la escala estándar (1 al 20).  
Al finalizar el ingreso, muestra el promedio general del grupo.

Tecnologías y Herramientas Utilizadas
* Lenguaje: C
* Control de Versiones: Git / GitHub
* Entorno de Ejecución: Docker
